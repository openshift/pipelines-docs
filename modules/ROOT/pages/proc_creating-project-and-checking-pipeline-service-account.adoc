= Creating a project and checking your Pipeline service account


[discrete]
=== Procedure

. Login to your OpenShift Container Platform cluster with your login and password:
+
----
$ oc login -u <login> -p <password> https://openshift.example.com:6443
----

. Create a project for the sample application:
+
----
$ oc new-project pipelines-tutorial
----

. The OpenShift Pipelines Operator adds and configures a Serviceaccount named `pipeline` that has sufficient permissions to build and push an image. This ServiceAccount is used by pipelineruns. Run the following command to see the `pipeline` service account:
+
----
$ oc get serviceaccount pipeline
----
