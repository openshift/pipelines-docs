Creating Custom Resources for the Pipelines Operator

Prerequisites

  • Ensure that the OpenShift Pipelines Operator is installed. For more information, see Installing the OpenShift Pipelines Operator.

Procedure

  1. In the OpenShift Pipelines Operator Subscription Overview screen, click the 1 installed link.

  2. In the OpenShift Pipelines Operator Overview tab, click Create New in the Provided APIs section to create an instance of the Openshift Pipelines install CRD. Note that you may need to refresh the screen, in case you do not see the section.

  3. In the Create Install custom resources editor screen, ensure that the namespace displayed is pipelines and click Create.

    The editor is pre-populated with definitions. You can configure the file by manually entering YAML or JSON definitions into the editor.

  4. The pipeline resources are now installed. In the OpenShift Pipelines Install tab, click on example-install to view the Openshift Pipeline resources installed by the operator.

  5. Check on OpenShift Pipelines Operator and resources status by navigating to Workloads → Pods.
